## 2. Conceito da Marca
O logotipo foi projetado para ser adaptável a diversos materiais de comunicação, mantendo uma identidade acolhedora e moderna.

* **Simbolismo (Cachorro)**: A marca utiliza a figura de um cachorro para representar o vínculo humano e uma relação afetiva agradável e de confiança.
* **Elemento Gráfico (Coração)**: Há um coração na ponta do nariz do cachorro como símbolo direto de afeto e cuidado.

---

## 3. Especificações Cromáticas (Cores)
A paleta é composta por quatro cores principais. O uso correto dessas cores é fundamental para a integridade da marca.

| Cor | HEX | RGB | CMYK |
| :--- | :--- | :--- | :--- |
| **Azul** | `#3F84AF` | 63, 132, 175 | 64, 25, 0, 31 |
| **Rosa** | `#E31A8B` | 227, 26, 139 | 0, 89, 39, 11 |
| **Preto** | `#000000` | 0, 0, 0 | 0, 0, 0, 100 |
| **Branco** | `#FFFFFF` | 255, 255, 255 | 0, 0, 0, 0 |

---

## 4. Tipografia
As fontes oficiais pertencem à família **Ubuntu**, uma fonte moderna, limpa e de alta legibilidade.

* **Principal (Para Títulos e Logo)**: Ubuntu Medium.
* **Secundária (Para Texto de Apoio)**: Ubuntu Normal.

---

## 5. Aplicações e Diretrizes
A marca possui variações de cor para garantir contraste em diferentes fundos (fundos claros, escuros ou coloridos).
